A cute cloud-shaped keyboard with a 3D-printed case!! The PCB has cute sketches on the silkscreen, and the PCB is controlled by a XIAO-ESP32-C3.

## ***how to use***
It's pretty straightforward to load up the board. First, solder the pieces together. Make sure to sandwich the keys together with the top of the case, putting the keys in from the top. After you're done soldering it, plug the ESP32 in, download and upload the python files into the folder that pops up. 

## ***inspiration***
I made this keyboard simply because I wanted a to make a personalized cute keyboard. It also helped me learn how to design PCBs. It's also always useful to have a keyboard the can perform quick shortcuts on the side.
